```java
public class Grade {
	public static void main(String[] args) {
			char grade;
			double avg =80;

			grade = getGrade(avg);
			// 우 입니다.
			System.out.println(grade + " 입니다.");

		}
	
		public static char getGrade(double avg) { 

			char grade; 
			if (avg >= 90) {
				grade = '수'; 
			} else if (avg >= 80) {
				grade = '우';
			} else if (avg >= 70) {
				grade = '미';
			} else if (avg >= 60) {
				grade = '양';
			} else {
				grade = '가';
			}
			return grade; 
		}

}
```

 *******

참고:[메소드 활용한 ](https://github.com/praybe/KOSMO/blob/main/eclipse-workspace/Method/src/StarPrint.java)

코드:
```java
public class StarPrint {
	public static void main(String[] args) {

		starPrint(5);

	}

	public static void starPrint(int num) { 

		for (int i = 1; i <= num; i++) {
			for (int j = i; j < num; j++) {
				System.out.print(" ");
			}
			for (int j = 1; j <= i * 2 - 1; j++) {
				System.out.print("*");

			}
			System.out.println();

		}

	}

}
```
